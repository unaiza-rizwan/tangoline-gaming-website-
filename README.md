## 🌟 Features

- **Animated Hero Section** — glowing text with fade-up entrance animations
- **Multi-page Navigation** — 7 pages with active link highlighting
- **Product Catalog** — PlayStation, Xbox & Nintendo consoles + games, each with price & buy button
- **Brand Filter Navigation** — jump to PlayStation / Xbox / Nintendo sections instantly
- **Category Cards** — hover lift effect with image overlays
- **Contact Form** — clean form with custom dark styling
- **Newsletter Subscription** — footer email capture
- **Fully Responsive** — mobile, tablet, desktop
- **Consistent Design System** — shared `shared.css` across all pages using CSS variables

## 🚀 Getting Started

No build tools, no dependencies — pure HTML/CSS.

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/tangoline-inc.git

# Open in browser
cd tangoline-inc
start tagoline.Inc.html   # Windows
open tagoline.Inc.html    # macOS
```

Or use **VS Code Live Server** for hot reload during development.

---

## 🌐 Deployment

Works on any static hosting — GitHub Pages, Netlify, Vercel.

### GitHub Pages (Free)
1. Push code to GitHub
2. Go to **Settings → Pages**
3. Source: `main` branch → `/root`
4. Your site: `https://username.github.io/tangoline-inc/tagoline.Inc.html`

### Netlify (One-click)
1. Drag & drop the folder to [netlify.com/drop](https://netlify.com/drop)
2. Live instantly — no config needed
